# pi-kimi-coding-plan-provider

Kimi Coding Plan provider extension for [pi](https://github.com/mariozechner/pi).

Provides access to Kimi K2.5 and K2 Thinking models via the Kimi Coding API with the Anthropic Messages format.

## Install

```bash
pi install pi-kimi-coding-plan-provider
```

Or add to your pi config:

```json
{
  "packages": ["pi-kimi-coding-plan-provider"]
}
```

## Auth

Use `/login` in pi and select **Kimi Coding Plan** to paste your API key.
